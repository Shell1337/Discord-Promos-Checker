# Discord Promos Checker
This Python script allows you to check 3m Discord Nitro Promos rapidly fast.

# Instructions:
- Put your 3m Promos in `input/promos.txt`.
- Put your Proxies in `input/proxies.txt`.
- Run the `main.py` file.
